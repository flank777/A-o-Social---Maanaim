# Ação Social - Arrecadação de Alimentos

Este projeto foi desenvolvido para auxiliar a ação social da igreja na arrecadação de alimentos, organização de doações e atendimento às famílias em situação de necessidade.

O sistema tem como objetivo facilitar o controle das cestas básicas, o cadastro de famílias assistidas, o registro de voluntários e a divulgação de formas de contribuição, como doações de alimentos e Pix.

Objetivo

Ajudar a igreja a organizar melhor o trabalho social, tornando o processo de doação mais simples, transparente e eficiente.

Funcionalidades
Cadastro de famílias atendidas
Controle de entrega de cestas básicas
Registro de alimentos doados
Organização de voluntários
Página com informações para doação via Pix
Contato via WhatsApp
Interface adaptada para celular
Apoio à divulgação da ação social


Este sistema foi criado com o propósito de servir ao próximo, apoiar famílias necessitadas e fortalecer o trabalho social realizado pela igreja.
